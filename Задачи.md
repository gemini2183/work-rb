# Задачи — общие

Клиентские задачи — в `Клиенты/<Имя>/Задачи.md`, не здесь. Здесь — только
инфраструктура, скрипты, база знаний, орг-вопросы. Сводный вид по всем
клиентам и общим задачам сразу — см. [[Обзор-задач]].

Тег `#задача` на каждой строке — нужен для запросов Tasks-плагина в
[[Обзор-задач]]. Даты в формате `📅 ГГГГ-ММ-ДД` (эмодзи "календарь") — Tasks
распознаёт это как дедлайн/scheduled автоматически.
