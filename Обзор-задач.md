# Обзор задач

Требует community-плагин **Tasks** (Settings → Community plugins → Browse →
"Tasks"). Сами задачи физически лежат в `Клиенты/<Имя>/Задачи.md` и корневом
`Задачи.md` — этот файл только агрегирует их живым запросом, ничего не
дублирует.
