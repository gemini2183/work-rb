# Инбокс

Сюда — любые мысли по ходу дня, без раздумий о структуре: решения, идеи,
задачи, наблюдения вперемешку. Формат — просто дата и текст, ничего сложнее.

По явному запросу ("разбери инбокс") агент разбирает записи и раскладывает их
по правильным местам (`Клиенты/<Имя>/Решения.md`, `Клиенты/<Имя>/Задачи.md`,
`База_знаний/Паттерны/`, `Бэклог_идей.md`, `Задачи.md` и т.д.), после чего
разобранные записи удаляются отсюда — инбокс остаётся пустым до новых заметок.
