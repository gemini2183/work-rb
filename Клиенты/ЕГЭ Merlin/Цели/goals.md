# Цели — ЕГЭ Merlin

Источник: вкладка "Metrika" общей Google-таблицы (см. [[../../../Инфраструктура|Инфраструктура]]).

## Макроцель (основная — используется для KPI 32 заявки / CPL 2500₽)

| Название | ID (Метрика) | Что считается | Ценность |
|---|---|---|---|
| Общая (с 11.10.21) | 222490461 | Основная цель-агрегатор в Метрике | заявка |

**Важно:** эта цель в Метрике покрывает не все заявки — звонки идут отдельно
через колл-трекинг (см. ниже), а не через Метрику напрямую.

## Колл-трекинг (звонки — НЕ через Метрику)

| Провайдер | ID цели | Источник данных |
|---|---|---|
| Callibri | 40404508 | Собственный API Callibri, `ct_goal_source=api` |

**Почему отдельно:** источник трафика "Яндекс.Директ: не определено" в Метрике
иногда не долетает до правильной атрибуции — часть звонков теряется, если брать
их только из Метрики. Поэтому по конвенции дашборда звонки берутся напрямую из
API колл-трекинга, а не из целей Метрики.

**Статус: скрипта для получения данных из Callibri у нас пока нет** — это
открытая задача, без неё цифра по заявкам неполная (см. [[../Решения|Решения]],
запись от 2026-07-16).

## Фильтр-условие для core-кампаний

`utm_content не содержит letnie` — кампания "летние курсы" (`N-709924126`)
должна исключаться из основного анализа (отдельный сегмент, не часть KPI на
июль на общих условиях). Проверено на снапшоте 11–15 июля: все 3 конверсии по
основной цели за этот период пришли именно с этой кампании — значит из
"основных" кампаний по Метрике конверсий не было вовсе (при том что звонки не
учтены).

## Микроцели (сигналы вовлечённости, не заявки)

Из выгрузки видно вспомогательные автоцели Метрики: "Автоцель: клик по номеру
телефона", "Автоцель: отправка формы", "Автоцель: заполнил контактные данные",
глубина скролла (scroll_10/25/50/90), время на сайте (30/60/90/120 сек),
посещения тематических страниц по предметам (ЕГЭ/ОГЭ: Биология, История,
Математика и т.д. — можно смотреть, какой предмет интереснее аудитории).
Статус: не используются для KPI, но могут быть полезны для анализа интереса
по предметам.
