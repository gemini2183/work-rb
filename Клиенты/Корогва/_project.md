# Корогва — бриф проекта

## Ниша / продукт

Украинский бренд одежды и флагов (corogva.com) — патриотическая/юмористическая
тематика (мотивы из "Батальон" / "Слуга народа", украинский фольклор). Онлайн-
магазин, e-commerce. Каталог:
- Флаги — коллекции 1.0–4.0, государственные, декоративные (от ~1650 ₴)
- Одежда мужская: футболки, лонгсливы, свитшоты, худи/зип-худи, шорты, штаны
- Одежда женская: футболки, зип-худи, шорты, штаны
- Разделы Новинки / Sale

Аудитория — молодёжь и взрослые украинцы, ценящие украинскую культуру и юмор.
Это объясняет разбивку аккаунта на два подпроекта (см. ниже): "Merchant Center"
— товарные объявления по фиду каталога (Shopping/PMax по товарам), "Google Ads"
— вероятно Search/PMax не по фиду (брендовые и тематические запросы).

## Технический контекст

Агентство Adwhite (Google Ads), не Redbird (Директ) — отдельная общая таблица
клиентов (см. [[../../Инфраструктура|Инфраструктура]]). Один аккаунт Google Ads
(client_id 845-234-6433, MCC login_customer_id 3023486398), разбитый на два
подпроекта по названию кампании, не по отдельному customer_id:
- "Корогва - Google Ads" — обычные кампании
- "Корогва - Merchant Center" — товарные кампании (название содержит "Sales"
  или "Merchant_")

Раньше конверсии собирались отдельно из GA4, трафик — из Google Ads API
(старая схема Adwhite). Сейчас переведено на единый источник — Google Ads API
целиком (трафик + конверсии одним запросом), GA4 не используется. См.
[[../../Скрипты/README|Скрипты/README]] и [[Решения]].
